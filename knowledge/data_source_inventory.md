# Data Source Inventory Template

Use this file as a checklist. Verify live URLs and terms before using any data.

## For every data source log

- Source name
- URL or access path
- Provider/owner
- Date accessed
- Version or snapshot
- License/terms
- Geographic coverage
- Time coverage
- Unit of observation
- Key variables
- Known breaks in measurement
- Merge keys
- Missingness notes

## Common source categories

- Official statistical agencies
- Administrative open data portals
- Central bank / fiscal / tax data portals
- Environmental monitoring data
- Transport and mobility data
- Platform/product telemetry
- Survey microdata
- Satellite or remote sensing data
- News/legal/policy timeline sources

## Data quality red flags

- Policy affects reporting or measurement.
- Treatment changes unit composition.
- Outcome definition changes at policy date.
- Missing data concentrated in treated units.
- Treatment timing is ambiguous or staged.
- Control units receive indirect exposure.
