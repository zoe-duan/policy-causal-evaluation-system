# Glossary

- **ATE**: Average Treatment Effect.
- **ATT**: Average Treatment Effect on the Treated.
- **CATE**: Conditional Average Treatment Effect.
- **LATE**: Local Average Treatment Effect for compliers under IV assumptions.
- **DID**: Difference-in-Differences.
- **Event study**: Dynamic DID design with leads/lags around treatment.
- **Synthetic control**: Weighted donor-pool counterfactual for treated aggregate unit.
- **DML**: Double / Debiased Machine Learning.
- **GRF**: Generalized Random Forests.
- **Policy learning**: Learning an assignment rule that maximizes welfare or outcome value.
- **SUTVA**: Stable Unit Treatment Value Assumption; no interference and well-defined treatment.
- **Overlap/common support**: Treated and control units have comparable covariate support.
- **Bad control**: Control variable affected by treatment or a collider.
