# 局限与伦理

1. **因果识别不是自动化产物**：系统可以推荐方法，但识别假设必须结合制度背景论证。
2. **相关不等于因果**：没有可信设计时，系统只能给出描述性分析或研究设计建议。
3. **政策评估要区分效果、成本、公平性和可执行性**：估计值显著不等于政策值得推广。
4. **异质性分析容易过拟合**：CATE、causal forest、policy learning 必须使用 honest splitting、holdout policy value 或外部验证。
5. **敏感人群与高风险政策**：教育、医疗、执法、福利等场景需要额外伦理审查和公平性检查。
6. **不要把 LLM 生成的文献和事实当真**：所有事实、法律、政策时间线、数据源必须核查。
