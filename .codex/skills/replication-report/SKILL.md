## Goal

Produce a policy evaluation report that is reproducible and honest about causal strength.

## Workflow

1. Read design card, analysis plan, outputs, diagnostics.
2. Label claim strength: strong, moderate, weak, or design-only.
3. Write results in assumption-conditional language.
4. Include evidence log for policy dates/data sources.
5. Include reproduction commands and package versions.
6. Ensure every numeric claim traces to a script or output file.

## Output files

- `evaluation_report.md`
- `evidence_log.md`
- `reproducibility_appendix.md`

## Guardrails

No fabricated citations. No unsupported causal language.
