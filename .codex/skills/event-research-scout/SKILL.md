## Goal

Find or vet policy events that are research-worthy and empirically feasible.

## Workflow

1. Define the domain: environment, labor, education, platform governance, industrial policy, health, tax, etc.
2. Use live web search for current or specific policy facts; prefer official documents, laws/regulations, statistical agencies, data portals, and reputable institutional sources.
3. Score events using `knowledge/research_event_bank.md`:
   - clear date
   - affected units
   - plausible comparison group
   - outcome data availability
   - enough pre-periods
   - limited spillovers
   - policy relevance
4. Return 3–5 candidate events with design sketches.
5. Do not invent event dates, policy names, or data source availability.

## Output

- Candidate event table.
- Best design for each event.
- Required source verification.
- Top recommendation.
