# DID / Event Study Checklist

- At least 3 useful pre-periods when possible.
- Treated and control units measured consistently.
- No obvious pre-policy shocks unique to treated units.
- No anticipation or define anticipation window.
- Treatment date encoded at correct unit/time level.
- Standard errors clustered at assignment level.
- Leads are reported; failure is discussed, not hidden.
- Robustness: alternative windows, controls, unit trends, donor subsets, placebo outcomes.
