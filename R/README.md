# R templates

Python scripts in `src/` provide runnable baselines. These R files are templates for production-grade estimators that are common in causal inference workflows.

Suggested packages:

```r
install.packages(c("fixest", "did", "Synth", "grf", "DoubleML", "mlr3", "data.table"))
```

Use these only after checking package versions and documenting the exact environment.
